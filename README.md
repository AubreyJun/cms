## 蓝科 - 企业网站  
RKCMS是开源免费的企业网站内容展示系统，使用PHP开发，基于YII2进行构建。

## 预览
![首页](http://backend.ranko.cn/uploads/file/cms.png)

## 安装步骤
1. 复制config目录下的dev_example目录到dev
2. 修改dev目录下的db.php中的数据库配置
3. 数据库导入SQL脚本,对应的脚本install目录下
